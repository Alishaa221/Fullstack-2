# React SPA Lab Experiments

This repository contains Single Page Application (SPA) experiments developed using React as part of the Full Stack Development lab.

## Experiments

### Experiment 1: Counter SPA
- A simple counter application
- Demonstrates use of React components and useState hook
- Updates UI dynamically without page reload

### Experiment 2: To-Do List SPA
- Task management application
- Allows users to add tasks dynamically
- Demonstrates state management and list rendering in React

## Technologies Used
- React
- JavaScript (ES6)
- HTML5
- CSS3
- Node.js

## How to Run
1. Navigate to the experiment folder
2. Install dependencies using `npm install`
3. Start the development server using `npm run dev` or `npm start`
